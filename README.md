# 城墨小说朗读

## 介绍

这是一款基于 Vue 的小说朗读应用，主要功能有：

- 章节切割
- 朗读记录保存
- 样式设置

## 开发

```bash
# 0. 克隆项目
git clone https://github.com/Pzc-Neo/vue-web-reader.git

# 1. 安装依赖
pnpm install

# 2. 启动项目
pnpm dev

# 3. 构建项目
pnpm build
```
